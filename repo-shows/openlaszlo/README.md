# A Repo Show: OpenLaszlo Reunion 🟦🧩

> The rich-internet-app platform that was years early — declarative, reactive, compiling to Flash
> *or* Ajax in 2002 — **revived with AI** and run live.

**The spark:** [David Temkin](../../characters/david-temkin/) recently used AI to rewrite the old
**OpenLaszlo** Java code so it **compiles and runs again** (per Don, relayed via
[Henry Minsky](../../characters/henry-minsky/)). That's pure Repo Show: resurrect a real toolchain
with AI, then bring the old **LZX** apps and demos back to life on it, live.

## The cast

- **[David Temkin](../../characters/david-temkin/)** — founder/CEO/CTO of Laszlo Systems; created OpenLaszlo; did the AI revival.
- **[Oliver Steele](../../characters/oliver-steele/)** — Chief Software Architect; designed **LZX**, wrote the compilers/runtime, launched the open-source project. (Marvin Minsky's son-in-law.)
- **[Henry Minsky](../../characters/henry-minsky/)** — Laszlo software architect; relayed the news. (Marvin's son.)
- **Don Hopkins** — worked on OpenLaszlo with all three; has old LZX apps + demos to resurrect.

- **Instance-first & reactive design** (Oliver's ideas) — what modern frameworks rediscovered.
- **The AI revival** — how the legacy Java toolchain came back; the making-of *is* the content.
- **Bring the apps back** — run Don's old LZX demos live on the revived toolchain.
