# A Repo Show with Patrick Hulin 🏢

> Binary → spec → tick-for-tick TypeScript. AI-assisted reverse engineering as a craft.

**Patrick Hulin** ([@phulin](https://github.com/phulin)) had an LLM (**Claude Code**, plus a
Unicorn-based emulator mocking ~195 Windows 3.1 APIs) reverse-engineer the original **SimTower**
binary into a complete simulation spec, then reimplemented it **tick-for-tick in TypeScript** —
byte-identical to the original on every tick, proven by replaying captured gameplay traces — and
made it real-time multiplayer. Live at **[towers.world](https://towers.world)**; open source as
**[tower-together](https://github.com/phulin/tower-together)** (MIT).
