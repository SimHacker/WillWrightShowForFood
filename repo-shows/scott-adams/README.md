## The show

In 1978 Scott Adams shipped **Adventureland** and, with it, a compact **data-driven adventure
engine** — each game a *database* of rooms, items, and verb/noun actions run by a small portable
interpreter, so one adventure ran on every home machine, a whole world in ~16K. That is the
**adventure compiler before it had a name** — the direct ancestor of Don's Terrapin **Logo
Adventure** (Logo REPL as the parser) and of Don's **adventure compiler** and **MOOLLM**.

A conversation across four decades of the same idea: adventure-as-data, constraints as craft,
the two-word parser vs. the LLM, and — since Scott is diving into **AI** now — where a founding
adventure author takes language models next. The best version: **Don builds adventure-compiler
tooling live** that Scott can actually use. Mutual seed exchange, real code in public.
