## The pitch

FujiNet started as "let's put WiFi on an Atari 8-bit" and became a **neutron star** pulling
siloed retro communities together — 5,000+ users, open hardware any vendor can build, disk
drives + modems + printers + an **`N:` network device** that makes HTTPS and TNFS look like
local peripherals to a 6502. Thomas's Feb 2026 VCF SoCal state-of-the-union covers RS-232
for MS-DOS, MSX/Intellivision bringups, cross-platform **FujiNet Battleship**, the **fish**
TNFS search shell, **Netstream** for Midi Maze, and an honest year of refactoring +
regression testing ahead.

This show is the **real-wire path** for the repo's Apple ][ stack, Retrocomputing Drive
challenges, and bridge episodes — complementing browser WASM emulators (POM2, AltirraSDL)
with **FujiNet-PC** (Fujisan, AppleWin+SPoverSLIP, MAME Becker port).

*"We should not argue to win. Rather, we should argue to give and gain insight."*
