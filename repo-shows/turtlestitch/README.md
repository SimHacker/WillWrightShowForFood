# Repo Show — TurtleStitch 🪡🐢

*Show seed. Source-grounded; creators and presenters credited, with consent.*

**TurtleStitch** turns block code into real machine embroidery: a fork of **Snap!** whose turtle is
the **needle** of a programmable embroidery machine — code on screen becomes thread on fabric.
Created by **Andrea Mayr-Stalder** (project leader) and **Michael Aschauer** (developer), built on
**Snap!** (Jens Mönig, Brian Harvey, UC Berkeley), with a community site on **Bernat Romagosa's**
Beetle Cloud. *(turtlestitch.org)*

## Two shows in one

1. **The tool** — the Logo → Scratch → Snap! → TurtleStitch lineage; turtle geometry as "a
   mathematics made for learning" (Papert); program a pattern live and stitch it on a real machine.
2. **The classroom moment** — why this is the real thing.

## The real thing (said plainly)

At Snap!Con (Barcelona), Don watched kids of every background light up making physical things from
code — including a young Black boy, **proud** of the sewing-machine project he'd programmed in
Snap!. Boys and girls combined what lazy stereotypes call "boy programming" and "girl sewing" with
**zero self-consciousness** — no embarrassment, no gatekeeping, no thought to gender, race, or
orientation.

We're promoting this **not** as a diversity slogan and **not** as a token gesture — that hollow,
performative version is exactly what makes people's eyes roll, and rightly so. We promote it because
it's the **genuine article**: constructionism doing exactly what Papert meant — children building
things that matter to them, across every line, because the work is real and fun. The concrete moment
carries it; no sermon required.

## Don's own stitch

At Snap!Con, Don programmed an embroidery machine with TurtleStitch — multi-color thread stitched
into felt, a circle of zigzag stitches tapering wide-to-narrow. *"It FELT SEW GOOD."*
