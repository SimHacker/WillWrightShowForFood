# Draft: Invitation to Will Wright (alternate concise-weird version)

**Subject:** Repo Show follow-up: the slower version of yesterday's whirlwind

---

Hey Will,

Great talking yesterday. I threw a lot at you fast, so here's the slower written
version: not a formal pitch, just a landing pad for the whirlwind.

The spark is your 1996 talk in Terry Winograd's Stanford UI seminar. I dug up
Stanford's recording, put it on YouTube, cleaned up the transcript, and have
been rewatching it closely. It's the talk where a student asks what you're
working on, you say "Oh, God...", load a SimCity file into Dollhouse, and start
describing the ideas that became The Sims.

That talk is the reason I went to Maxis.

I'm starting something I'm calling a **Repo Show**: a live conversation where
the stage is a public GitHub repo. We talk through source material, old demos,
design memories, half-finished ideas, and then I use the repo and AI tools to
turn some of those ideas into working artifacts while the conversation is still
alive. Not a podcast where the ideas evaporate afterward: a show with receipts,
source files, transcripts, prototypes, and follow-through.

I'd love you to be the first guest.

No assigned topic. You're the topic. We can start with the 1996 video and follow
whatever threads you want: SimCity, Dollhouse, The Sims, Proxi, user-created
content, data portability, objects that advertise behavior, why some simulations
work and others don't, or whatever you're thinking about now.

One arc I think would be especially fun: the 1996 talk looked back at SimAnt,
SimEarth, and SimCity 2000, then looked forward to Dollhouse. Thirty years after
that preview, and twenty-six years after The Sims shipped, we can finally do the
long-delayed Sims 1 postmortem, using your own 1996 words as the seed: what you
nailed, what surprised you, what you'd call differently now.

The crown jewel is still **data portability**: characters, homes, objects,
memories, and stories moving between worlds. You described that train-set hobby
model in 1996, and it still feels unfinished. I'd love to sketch a little bridge
with you, even if it's just a toy prototype at first: Proxi ideas, Sims ideas,
Micropolis/Simopolis ideas, all talking to each other instead of living in
separate silos.

The AI angle is practical. I use it as a fast pair programmer, archivist, and
transcript-wrangler while we talk. The point is still the people and the ideas;
the AI just helps turn them into artifacts before they evaporate.

Format is low-pressure: live, friendly, no prep, no slides. I'll handle the
repo, clips, transcript, and setup. We can decide together what is public, what
is recorded, and what stays private.

If it sounds fun, I'll send the repo link and we can find a time.

— Don

*(the User Interface Flower Child)*
