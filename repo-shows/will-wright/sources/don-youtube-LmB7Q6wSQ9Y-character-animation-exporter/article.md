# The Sims 1 Character Animation Exporter — MaxScript + Ocean Quigley

*YouTube: https://www.youtube.com/watch?v=LmB7Q6wSQ9Y*

## What it shows

The **VitaBoy / skeleton animation export** path Don built for The Sims 1 — MaxScript front-end, C++ Sims animation library back-end, shaped by watching **Ocean Quigley** rig and animate in Max daily.

## Design lesson (Don → Fei Ou, AIFoundation 2020)

> I had to learn how real artists used the tools everyday… instead of imposing a pre-conceived design from the perspective of a programmer.

That same pattern drives Micropolis Home: tools shaped around Heather/Steve's SimFreaks/SimSlice workflows, not generic CMS assumptions.

## Technical notes

- **John Wainwright** — MaxScript creator; Don knew him from Kaleida **ScriptX**  
- Note tracks → validated segments → SCM/database → batch export to game format  
- Foundation for **SimShow** (pre-ship fan skins) and in-game character pipeline
