# SimCity 3000 scrapped 3D build — preservation hub

## Story (short)

E3 1997 showed a **3D SimCity 3000** prototype. The codebase was abandoned (~Nov 1997); gameplay additions were salvaged for the shipped **2D isometric** SC3K. Maxis ran out of money; **EA acquisition** let the team finish **The Sims** and restart SimCity as 2D. Don argues it should have been **SimCity 2001** — realistic ship window.

**Guru** (lead designer on the 3D tangent) wrote an infamous design doc — sincere hubris, 15 years ahead of hardware. **Ocean Quigley**: Guru believed it, wasn't joking.
