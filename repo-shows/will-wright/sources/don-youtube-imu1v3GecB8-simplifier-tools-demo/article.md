# Transmogrifier, RugOMatic, ShowNTell, Simplifier, and SliceCity — demo walkthrough

*YouTube: https://www.youtube.com/watch?v=Imu1v3GecB8 · Referenced in Don's 2020 Unity/AIFoundation portfolio mail and old-email harvest.*

## What this video is

One-take tour of **The Sims 1** fan-tool era:

| Tool | Purpose |
|------|---------|
| **Transmogrifier** | Player-authored objects — IFF export pipeline |
| **RugOMatic** | Custom rugs / floor treatments |
| **ShowNTell** | Object preview / catalog (ActiveX era ancestor of Maxis catalog previews) |
| **Simplifier** | **The Sims Online** dual-client bot — screen scrape + input injection for maze/money tasks |
| **SliceCity** | Steve Alvey's **SimCity inside The Sims** (SimSlice) |
