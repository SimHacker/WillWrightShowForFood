## Ocean Quigley → MaxScript exporter

1997: Ocean walked Don through **3D Studio Max**, Biped, Character Studio — "insanely complicated" but essential for designing the artist-facing pipeline.

Design choices Don highlights:

- **Max note tracks** mark animation/skeleton/mesh boundaries
- Validation against content DB + **automatic source control** check-in/out
- **Batch export** without manual dialog drudgery every iteration
- Player tools insight → **SimShow** (characters) and **Transmogrifier** (objects) before retail ship

MaxScript sample (Turbo-Deluxe UI snapshot):

https://donhopkins.com/home/archive/maxscript/maxis-maxscript.ms.txt

John Wainwright connection: Kaleida **ScriptX** → **MaxScript** architect — Don used MaxScript + C++ plugins calling Sims animation library directly.
