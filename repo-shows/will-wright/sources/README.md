threads, Wayback captures. This is the **story we tell in public**: SimCity open source,
OLPC, Transmogrifier, The Sims birth, procedural rhetoric, preservation.

## Start here — the centerpiece

**[1996-04-26-winograd-interfacing-to-microworlds/](1996-04-26-winograd-interfacing-to-microworlds/)**  
Will's Stanford CS547 talk — SimEarth/SimAnt/SimCity postmortems, then the **Dollhouse** demo
that became The Sims. Don was in the room; preserved transcript + Medium article + 167 figures.

**[2008-02-10-soul-of-the-sims/](2008-02-10-soul-of-the-sims/)**

on this same motive machinery.
