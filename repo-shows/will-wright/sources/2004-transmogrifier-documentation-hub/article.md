# The Sims Transmogrifier Documentation

*By Don Hopkins · Wayback Feb 2005 · Links on original site: Home · Tutorial 1 · Glossary · License*

---

## What Transmogrifier is

A tool to **clone** an existing Sims object and edit its **name, price, description, and sprite images**. Graphics export as **BMP** files for Photoshop / Paint Shop Pro, then import back.

Like **SimShow**, scope is intentionally limited for ease of use.

Download: `thesimstransmogrifier.com/TransmogrifierDownload.htm` (Wayback era).

---

## What Transmogrifier is NOT

| Limitation | Detail |
|------------|--------|
| **Support** | **NOT SUPPORTED** by Maxis or EA — do not ask EA/Maxis for help, features, or bug fixes |
| **2D editor** | Writes BMPs; you edit in external paint tools |
| **3D editor** | No polygon meshes — pre-rendered 2D sprites only |
| **Behavior editor** | Behaviors are **SimAntics** in **Edith** (special Sims build with editors) — separate future release, unsupported |
| **Industrial pipeline** | Not for high-volume professional content |

---

## Cannot edit (2004)

Slots, tree tables, sounds, headlines, conversation topics, water tiles, floors, walls, wall styles/patterns, cutaway/thick walls, UI bitmaps, raw resources, character bodies/skins/animation, house properties, relationships, family histories.

Future tools planned to extend **XML** export/import for more data types.
