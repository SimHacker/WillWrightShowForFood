# Dumbold Voting Machine

Interactive **agitprop** Sims 1 object (2004 election cycle). Procedural rhetoric as executable multimedia.

## Primary source (full article)

- **[Dumbold Voting Machine for The Sims 1](../../sources/2018-04-24-dumbold-voting-machine/medium-article.md)** — clean republishable text extracted from Medium
- Original: [donhopkins.medium.com](https://donhopkins.medium.com/dumbold-voting-machine-for-the-sims-1-3e76f394452c)

## High concept

A puzzle you can "hack into" — simulates real e-voting failure modes (security holes, audit log tampering, "Monkey" debug mode) while showing illustrated pop-ups about Diebold, EFF, DMCA §512(f), etc.

Free speech rendered as **SimAntics + dialog UI + pie menus**.
