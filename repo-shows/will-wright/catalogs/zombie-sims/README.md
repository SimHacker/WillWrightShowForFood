# Zombie Sims

**Heather Alvey + Steve Alvey** — current magnum-opus expansion pack for The Sims 1.

## What

Artisanal zombie suites: IFF objects, SimAntics behaviors, interchangeable skins. Still actively shipping — not archive nostalgia.

- Hosted on: [Micropolis Home](../micropolis-home/)
- Sister catalogs: [SimFreaks](../simfreaks/) · [SimSlice](../simslice/)
