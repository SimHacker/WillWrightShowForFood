# SimFreaks

**Heather Alvey** (née Castello) — ~26-year Sims custom-content brand.

## What

Original object suites, sophisticated SimAntics, themed collections, and community leadership.
One of the deepest artisanal catalogs in Sims 1 history.

- Storefront: [simfreaks.com](https://www.simfreaks.com/)
