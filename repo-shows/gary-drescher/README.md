# A Repo Show with Gary Drescher 👤

> Made-Up Minds, remade — Piaget you can run, now with a language model for grounding.

**Gary Drescher**'s *Made-Up Minds* (MIT Press, 1991) built the **schema mechanism**: an agent that
learns about its world from raw experience by discovering **schemas** (context → action → result),
using **marginal attribution** and **synthetic items** — Piagetian constructivism made to run. His
*Good and Real* (2006) brings the same rigor to choice, determinism, and ethics.

## What we'd get into

- **The schema mechanism from first principles** — what it did, and what held up.
- **Made-Up Minds, remade** — give it an **LLM** for the grounding it once learned from scratch, and
  build it live on [Henry Minsky](../../characters/henry-minsky/)'s **Python schema factory**.
- **The lineage** — [Piaget](../../characters/jean-piaget/) → schema mechanism →
  [Leela's](../../characters/steve-kommrusch/) Piaget-schema learning.
- Optional philosophy turn from **Good and Real**.
