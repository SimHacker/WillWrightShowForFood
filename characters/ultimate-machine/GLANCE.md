🔌 **The Ultimate Machine** *(fictional bot)* — a box whose only function is to switch itself off. Real origin: Marvin Minsky's 1952 design (he called it the "ultimate machine"); Claude Shannon built the first; Arthur C. Clarke found it "unspeakably sinister."
