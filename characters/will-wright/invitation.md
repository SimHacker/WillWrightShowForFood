## Invitation

We would be honored if you would join a **Repo Show** — a conversation whose stage is
[this public repository](https://github.com/SimHacker/WillWrightShowForFood), following
through to working code and shared technique. **Show, don't tell.**

**You would be the first guest — and there is no assigned topic. You are the topic.**
We would start from your **1996 Stanford talk** to Terry Winograd's class (Dollhouse /
data portability) and follow wherever you want to go.

### Why this repo

- **The arc:** In 1996 you looked back at SimAnt, SimEarth, SimCity 2000, then previewed
  The Sims. We can finally do the long-overdue **post-mortem on The Sims** — same room,
  decades later.
- **The crown jewel:** **Data portability** — characters, homes, objects, memories moving
  between worlds. We want to build toward that with you, in public.
- **No homework:** No prep deck. Warm conversation; ideas can become running code on stream.
- **Reincarnate Slats (optional segment):** **Slats** — the robot brain Don programmed at your
  **Stupid Fun Club**, who appeared in the One Minute Movies (*Servitude*, *Empathy*). Don still
  has the code. If it sounds fun, we'd pursue it as a live **RoboResurrection** quest on the show:
  read the old code together, revive him, call-in sidekick, reprogram, iterate. Stupid Fun Club
  energy — not a required topic. Details: [slats-reincarnation.yml](https://github.com/SimHacker/WillWrightShowForFood/blob/main/repo-shows/will-wright/slats-reincarnation.yml).
