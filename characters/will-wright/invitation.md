## Invitation

We would be honored if you would join a **Repo Show** — a conversation whose stage is
[this public repository](https://github.com/SimHacker/WillWrightShowForFood), following
through to working code and shared technique. **Show, don't tell.**

data portability) and follow wherever you want to go.

- **The arc:** In 1996 you looked back at SimAnt, SimEarth, SimCity 2000, then previewed
  The Sims. We can finally do the long-overdue **post-mortem on The Sims** — same room,
  decades later.
- **The crown jewel:** **Data portability** — characters, homes, objects, memories moving
  between worlds. We want to build toward that with you, in public.
- **No homework:** No prep deck. Warm conversation; ideas can become running code on stream.
