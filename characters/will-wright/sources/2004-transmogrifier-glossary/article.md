# The Sims Transmogrifier Glossary

*By Don Hopkins · Wayback capture Mar 2005 · "Work in progress, not yet complete nor alphabetized" (original site).*

---

## Core concepts

| Term | Definition |
|------|------------|
| **Object** | Most catalog/buyable/world entities; packaged in **Object Files** with unique **Object ID** |
| **Object ID** | 32-bit: upper 16 bits = **Magic Cookie**, lower 16 = random uniqueness |
| **Magic Cookie** | 16-bit producer ID; 65535 available; honor-system registry prevents ID collisions |
| **Object File** | `.iff` / `.spf`; may contain multiple objects; filename independent of product name |
| **Far File** | `.far` archive (like zip, different format) |
| **Single / Multi Tile Object** | 1×1 vs multi-cell; multi-tile uses **Master** (catalog abstract), **Lead** (world tile), **Slave** tiles |

---

## Catalog & graphics

| Term | Definition |
|------|------------|
| **Product Name / Price / Description** | Editable in Transmogrifier Edit Object window |
| **Catalog Icon / Popup / Speech Icon** | Auto-generated by The Sims from draw groups |
| **Sprite** | 2D image: pixel palette BMP + Z-buffer BMP + alpha BMP per frame |
| **Graphical State** | Switches which sprites/draw groups are active; SimAntics-driven |
| **Zoom** | Small, Medium, Large |
| **Rotation** | Four views; may flip symmetric sprites to save memory |
| **Symmetry** | Bilateral (2 images) or radial (1 image) |

---

## Transmogrifier file formats

| Term | Definition |
|------|------------|
| **BMP File** | 8-bit palettized or 24-bit RGB Windows bitmap |
| **XML File** | Export/import manifest linking BMPs + export settings for round-trip |
| **Export Whizzer** | Dialog controlling export granularity (colors-only vs shape, zooms, channels) |

## Related tools (documented, mostly unreleased to fans)

| Tool | Role |
|------|------|
| **Transmogrifier** | Clone objects; edit text; export/import sprite BMPs |
| **SimShow** | Character skins and animations (TODO on original site) |
| **Edith** | SimAntics behavior programming; special Sims build with editors (future separate release) |
| **SimAntics** | Visual language controlling object/character behavior |

People are rendered live in 3D — not editable via Transmogrifier sprites.
