# SimRefinery — finding it, playing it, and the archivist who dug it up

**Status:** Link hub + article backlog. **Do not host** the game binary in git — point to archives and Phil Salvador's scholarship.

## The story (short)

Phil Salvador (The Obscuritory / VGHF) researched **Maxis Business Simulations** and the lost **SimRefinery** training prototype for Chevron. An Ars Technica reader surfaced a 3.5" floppy; the disk was uploaded to the Internet Archive (June 2020). Salvador's analysis is the best guide to what the incomplete build actually does.

## How to find and play (when IA is up)

1. Open the IA item in a browser (embedded DOSBox) **or** download files locally.
2. After install, at the prompt: `intro` (tutorial) or `simref` (game).
3. Local alternative: `intro.bat` via DOSBox; expect incomplete UI — many buttons non-functional.
4. Read Salvador's **Tour Book** commentary — in-game tutorial alone is insufficient.
