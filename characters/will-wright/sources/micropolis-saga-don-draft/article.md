## OLPC open-source path (Don's summary)

1. **John Gilmore** suggested SimCity Classic for OLPC → Don asked **Will Wright**  
2. Will liked it → forwarded to EA execs → **legal favorable**  
3. **Chuck Normann** ("Chuck Norman" in draft) shepherded internally — "favors due" for slack to push project  
4. **One executive against**, later changed mind (**Don to resolve: name**)  
5. EA Legal approved → **change list + GPL v3 + copyright notices**  
6. Stipulation: call it **"SimCity"** only after **EA QA** approved build  
7. Don fixed X11/TCL/Tk port for XO, applied EA graphics, set up **Linux VM under Windows** for EA QA (unfamiliar with Linux QA)  
8. One QA-approved **SimCity** OLPC release → renamed **Micropolis** (Will's original working title) to avoid per-change EA approval  
9. C++ overhaul, Python, PyGTK desktop UI, OpenLaszlo/Flash web server, Java port on Google Code
