# Ted Selker

[Invitation](invitation.md) · [Show seed](../../repo-shows/ted-selker/README.md) · [CARD](CARD.yml)

The man behind the **TrackPoint** (that red nub between G/H/B that keeps your hands on the keys). He ran IBM Almaden's **NPUC — New Paradigms for Using Computers** workshops — the same "new paradigms" venue where **Don demoed Multiplayer SimCity** — championed **considerate computing** at the MIT Media Lab, and co-directed the **Caltech/MIT Voting Technology Project** after the 2000 election. Input efficiency and attention-respecting design are right in this show's wheelhouse.
