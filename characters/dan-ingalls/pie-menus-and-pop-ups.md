## The family resemblance

| | **Pop-up menu** (Dan, Smalltalk) | **Pie menu** (Don) |
|---|----------------------------------|---------------------|
| **Shape** | Vertical list at cursor | Radial wedges around cursor |
| **Gesture** | Move + release on item | Direction + release (Fitts-friendly) |
| **Lineage** | Invented with BitBlt-era Smalltalk UI | CHI'88 with Mark Weiser; NeWS, SimCity, The Sims |
| **Idea** | Choice where you're pointing | Same — direct manipulation at the cursor |

Alan Kay introduced Don to Dan via the **Fabrik** paper (2018) — visual dataflow and direct manipulation as kindred ideas, not just menu geometry.
