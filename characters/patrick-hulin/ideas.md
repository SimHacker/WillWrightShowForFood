## What Patrick has done

Patrick Hulin (GitHub @phulin) reverse-engineered the original SimTower Windows binary with an LLM (Claude Code + a Unicorn-based emulator mocking ~195 Win 3.1 APIs), wrote a complete spec of its simulation, and reimplemented it tick-for-tick in TypeScript — byte-identical to the original on every tick, proven by trace tests — then added real-time multiplayer (Cloudflare Workers) and a build-a-grid UI. Live at towers.world; open source as tower-together (MIT).
