# Don Hopkins — portrayal data

Supporting YAML for the host's self-portrayal. Don is self-sovereign (consent level 1); these files
shape how Don is represented and referenced across the repo.

- [`voice.yml`](voice.yml) — Don's voice/style (tone, idioms, how to write *as/about* Don).
- [`heroes.yml`](heroes.yml) — the people and traditions Don draws on and credits.
- [`presentations.yml`](presentations.yml) — talks, demos, and public presentations.

See [`../README.md`](../README.md) and [`../CHARACTER.yml`](../CHARACTER.yml).
