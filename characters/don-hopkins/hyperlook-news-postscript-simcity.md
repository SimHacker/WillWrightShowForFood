# HyperLook / NeWS / PostScript — SimCity's first Unix face

**DUX Software (1991–1992):** SimCity on **NeWS** with **HyperLook** — PostScript UI, pie menus, and a **PostScript tile renderer**. Parallel to the official Sun X11 port; ancestor of every later SimCity Unix UI.

## Thread

- **Arthur van Hoff** — HyperLook (NeWS HyperCard), GoodNeWS lineage
- **Don** — showcase package, cooperative multiplayer ideas on workstations
- **Imaging braid starts here** — Porter/Duff stencil-paint in the window server
