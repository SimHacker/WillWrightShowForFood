# MOOLLM — The Sims for your filesystem (flagship talk)

**One liner:** A MUD/MOO where rooms are real directories, YAML files are objects with state,
skills are inheritable prototypes, and an LLM navigates the world with you — wired to the life
you're actually living.

## Spine (telescoping)

- `ls` as entering a room — filesystem is already a world model
- Semantic YAML, skills as prototypes, robust-first degradation
- Live demo branch: home automation twin **or** adventure compiler **or** career memory palace
- Nested rooms: DreamScape @ WWDC 1995, iLoci 2008, pie menus @ 40, open-sourcing SimCity…
