## What this is

A **Repo Show**: a live conversation whose stage is *this repo*, which **follows through
to working code**. Part of **Micropolis Class** — go to the people who invented this stuff,
capture credited ideas in public, let AI implement them on stream. **Show, don't tell.**

**Will Wright is the first guest — topic-less by design.** We orbit his 1996 Stanford
talk (Dollhouse → The Sims preview), then follow wherever he wants. The crown jewel:
**data portability** — characters, homes, objects, memories moving between games
(Proxi ↔ Sims ↔ …).

## MOOLLM plugin world

This repo is a **full-fledged MOOLLM microworld**. Open it alongside
[`SimHacker/moollm`](https://github.com/SimHacker/moollm) in Cursor (or your orchestrator)
and it composes — skills, characters, rooms, yaml-jazz indexes. See [`kernel/moollm-plugin.yml`](kernel/moollm-plugin.yml).

Someday: Cursor sponsorship. Someday: other orchestrators + our own. The export/sync plan
lives in [`process/sync-with-donhopkins.yml`](process/sync-with-donhopkins.yml).
